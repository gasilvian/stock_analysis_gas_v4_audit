# Real-source population area

This folder is intentionally separate from synthetic construction data.

Use it for versioned real/curated inputs only:

- `universe/` — curated ticker universes by market/region/industry.
- `rates/` — official or curated bond-yield and ERP sources.
- `fx/` — EOD FX rate files.
- `payloads/` — yfinance live payloads or manually curated payloads.
- `snapshots/` — recorded yfinance raw snapshots.
- `manifests/` — run manifests generated by `populate-real-sources`.

Template or synthetic files are not production-ready. The production-readiness gate checks this.
